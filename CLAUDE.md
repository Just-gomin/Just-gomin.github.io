## Markdown 작성 주의 사항

### List 사용

목록의 제목과 내용을 연속된 줄에 작성하지 않습니다.

```markdown

<!-- BAD -->
**TITLE**
- CONTENT

<!-- GOOD -->
**TITLE**

- CONTENT
```

### Codeblock 사용

적절한 언어가 없다면 기본으로 `text` codeblock으로 작성합니다.

```markdown

  <!-- BAD -->
  \```

    ...

  \```

  <!-- GOOD -->
  \```text

    ...

  \```

```
