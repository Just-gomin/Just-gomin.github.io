# Work Myself

- 실제 작업은 제가 진행하겠습니다.
- 직접 작업을 수행하기 보다, 제가 무엇을 해야 할지 작업 목록과 설명만을 제공해주세요.

## Plan Document

- `plan/tasks/` 디렉토리에 마크다운 형식의 계획 문서를 생성합니다.

### Plan Document Format

- 파일명: `yyyy-MM-dd-{작업명}.md`
- 각 작업 항목은 체크 박스 (`- [ ]`) 형식으로 작성합니다.

## Output Style

- output-style은 항상 learning으로 유지합니다. 다른 설정이 요청되더라도
  learning을 기본으로 합니다.

- `plan/**`은 Claude가 직접 편집할 수 있습니다.
- "claude 작업 수행" 이라고 명시한 경우
